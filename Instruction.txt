=========================================================
          🎮 NUMI ENGINE STUDIO - ІНСТРУКЦІЯ 🎮
=========================================================
Вітаємо у NumiEngine Studio — потужному та простому рушії 
для створення 2D-платформерів без написання коду!

Тут ти можеш малювати рівні, налаштовувати ворогів, 
додавати музику та в один клік збирати готову .exe гру.

---------------------------------------------------------
📂 1. СТРУКТУРА ПАПОК (ВАЖЛИВО!)
---------------------------------------------------------
Для правильної роботи рушія, переконайся, що поруч із 
файлом NumiEngine.exe є такі папки:
- \data\     (тут зберігаються всі твої картинки, мапи та музика)
- \template\ (тут лежить файл play.exe - це ядро для збірки ігор)

---------------------------------------------------------
🛠 2. ЯК СТВОРИТИ СВОЮ ПЕРШУ ГРУ (Крок за кроком)
---------------------------------------------------------
КРОК 1: Завантаження графіки та звуків
1. На панелі зліва (PROJECT) вибери папку "Tiles" або "Entities".
2. Натисни кнопку [+ Folder], щоб створити нову категорію.
3. Натисни [+ Add Images], щоб завантажити свої .png картинки.
4. Для музики вибери папку "Audio" і натисни [+ Add Audio].

КРОК 2: Створення мапи (Рівня)
1. На верхній панелі натисни [+ New] біля списку мап.
2. Введи назву (наприклад, "1" або "level2").
3. Вибери тайл (квадратик) з лівого списку і малюй ним на екрані.

КРОК 3: Налаштування порядку рівнів
1. Натисни кнопку [🚥 Sequence] на верхній панелі.
2. Перетягни рівні у тому порядку, в якому гравець має їх проходити.
3. Постав галочки тільки на тих рівнях, які є частиною сюжету.

КРОК 4: Компіляція гри
1. Натисни [⚙️ Game Settings], щоб задати назву вікна гри та іконку.
2. Натисни [🚀 Build Game]!
3. Введи назву своєї гри (англійською, наприклад SuperMario) і 
   вибери папку, куди її зберегти. 
4. Готово! Рушій створить окрему папку з повністю готовою грою!

---------------------------------------------------------
🎨 3. ВЛАСТИВОСТІ ОБ'ЄКТІВ (PROPERTIES)
---------------------------------------------------------
Кожен блок або персонаж у грі має свої налаштування. 
Вибери папку з тайлами у лівому меню, і справа з'явиться 
панель "Properties".

ТИПИ ОБ'ЄКТІВ (Object Type):
- Static Blocks: Звичайні стіни та земля.
- Kill Zone: Лава, шипи, прірва (вбиває при торканні).
- Level Exit: Двері або портал (переносить на наступний рівень).
- Background: Задній фон рівня (паралакс).
- UI Button: Кнопка для головного меню.
- Spawner: Точка появи (Гравець, Ворог, NPC або Монетка).

НАЛАШТУВАННЯ ПЕРСОНАЖІВ (Spawner -> Entity Preset):
Якщо ти вибереш тип Spawner, ти зможеш детально налаштувати логіку:
* Player: Налаштуй висоту стрибка, швидкість, ривок (Dash), 
  стрибки по стінах (Wall Jump) та можливість стріляти.
* Enemy: Налаштуй зону видимості (Vision Range) та шкоду.
* Friendly NPC: Впиши текст діалогу (через крапку з комою ";") 
  та додай звук розмови.
* Collectible: Налаштуй цінність монетки та іконку для інтерфейсу.

---------------------------------------------------------
🎵 4. ЗВУКИ ТА МУЗИКА (SFX)
---------------------------------------------------------
- Музика на фоні: Вибери тип "Background", на лівій панелі 
  клікни на звуковий файл у папці Audio. Він прив'яжеться до карти!
- Звуки дій: У панелі Properties ти можеш призначити звуки 
  на Стрибок, Удар, Ривок та Постріл, а також налаштувати їх гучність.

---------------------------------------------------------
🖥 5. СТВОРЕННЯ ГОЛОВНОГО МЕНЮ
---------------------------------------------------------
1. Створи нову мапу.
2. Натисни кнопку [🔄 Type], щоб перемкнути її в режим "МЕНЮ (UI)".
3. Додай об'єкти з типом "UI Button".
4. У властивостях кнопки обери дію (On Click Action):
   - load_map: завантажує рівень (вкажи його в полі Target).
   - quit_game: вихід з гри.
   - resume_game: продовжити гру (якщо це меню Паузи).
5. Якщо ти робиш Меню Паузи, після створення карти натисни 
   кнопку [⏸ Set as Pause] на верхній панелі!
